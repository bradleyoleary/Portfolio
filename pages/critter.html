Critter

Project Overview

For this project, I was tasked with creating a clone of Twitter. Critter was an independent project I worked on when I attended the Full-Stack Web Development program at Concordia University.

Overall, this project was a great learning experience to hone in on my front-end development skills.

Explanation & Objective

An API was provided and my task was to develop the front-end of the application. Some of the requirements included:

• Creating the UI from a provided design

• View a home feed

• Ability to post a tweet

• Ability to like tweets

• View a profile page

• View a single tweet
